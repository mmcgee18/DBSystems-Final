# Culinary Compass 🧭
Georgia State University | Database Systems (CSC 4710) | Spring 2026 <br>
<hr>
Culinary Compass is a comprehensive, open-source web application designed to simplify kitchen management. It serves as a centralized hub for tracking personal recipes, managing pantry ingredients, and organizing weekly meal plans. <br>
<hr>
🚀 Project Overview<br>
This project was developed as a final requirement for the Database Systems course at GSU. It demonstrates full CRUD functionality, complex SQL querying, and seamless integration between a Python/PHP backend and a MySQL database. <br>
Core Features <br>
Recipe Repository: Store, edit, and categorize your favorite culinary creations. <br>
Smart Pantry: Track ingredient quantities in real-time with low-stock alerts. <br>
Meal Planner: Schedule your week and automatically generate shopping lists. <br>
Search & Filter: Find recipes based on available ingredients or dietary preferences. <br>
<hr>
🛠️ Tech Stack <br>
Backend: PHP & Python <br>
Database: MySQL (via MariaDB) <br>
Management: phpMyAdmin & XAMPP <br>
Interface: HTML5, CSS3, JavaScript <br>
